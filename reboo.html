Reboo.org

Reboo is the main project I’ve worked on since joining the Our Cities Network. The project’s goal is to unify all of the 6 web apps they had previously develped in one single, robust platform for activism and causes for urban citizens.

Platform architecture

The first step in creating Reboo.org was getting everyone onboard with what it sould do in the first place. The activists have so many technological need that it was very, very hard to define the scope that it should be contained in. We wanted, most importantly, to enable users to build a personalized website to tell a story about their cause; secondly, to train them, through tutorials and other pedagocical tricks, how to build an effective campaign that can effectively make a change in the world; and thirdly, to connect activists and their causes so they can work together and share knowledge.

Sketching and writing

Sketching is an exercise that, for a lot of people, is something that can be taken for granted, or that is meticulously done to impress in Dribbble shots, not really as a tool for helping product development.

To me, skething is an essencial step to gain an understanding of the problem. The pages below are ugly, messy, disposable drawings that are not made in order to represent this or that screen or functionality, but quickly jotted down during conversations and ideations as a tool for thinking.

Wireframing

There was a lot of trial and error involved in making this product. We needed to make sure not to start developing too early, in order to avoid having to throw everything away if scope changed, so a lot of time was allocated to drawing low-fidelity wireframes that were reviwed daily by the team.

Style Guides

It’s very easy to start getting inconsistencies in the interface styles after some design iterations. We’ve decided, then, that keeping an evolving style guide file can be a good way of getting things organized and neat.

We’ve borrowed a lot from Google’s Material design guidelines, which are especially useful when building forms, something that Reboo users have to deal a lot with. Apart from that, we tried to stay simple and pack some personality, that also comes across through carefully written interface copy.
