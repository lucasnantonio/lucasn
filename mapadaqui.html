Mapa Daqui

Mapa Daqui is my graduation thesis project, in which I explored some interests like urban mobility, activism, and collaborative processes. It’s a collaborative, open-source, low-cost, bottom-up wayfinding system for pedestrians in Brazilian cities.

Go to mapadaqui.org

An open-source platform for wayfinding

Soon, it became clear that the ideal system would enable anyone, anywhere to create, download, print, and install their own signs. The result is an open source, free to use platform that does just that.

Process

Mapa Daqui was a project developed during a whole year. It combined design methods and lean startup methods, in order to create a service with good user understanding and empathic basis, with speed and multiple iterations.

Storyboarding was one of the tools used during ideation to come up with the user stories the service would need to provide. These stories were then prioritized and implemented in steps, with goals, metrics and tests in each iteration.

Skething and rapid prototyping were essential tools for coming up with the final sign design. It is based on well-known wayfinding systems like Legible Longon and Walk NYC, but adds a layer of participation, collaboration, and interactivity to the pedestrians.
